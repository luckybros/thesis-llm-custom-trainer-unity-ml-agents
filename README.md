# LLM Custom Trainer for Unity ML-Agents

This repository contains a custom plugin for the Unity ML-Agents toolkit that implement a LLM+RL training based, treating LLMs as a prior action distribution and integrating them into the RL framework.

---
## Requirements:
- ml-agents, please install it following the official documentation at https://github.com/Unity-Technologies/ml-agents/blob/develop/docs/Installation.md
- Unity

---
